# ScanKey V1A1 (Expo)

## Requisitos
- Node 18+
- Expo (se usa vía `npx`)
- Expo Go en el móvil (Android/iOS) o emulador

## Instalar y ejecutar
```bash
git clone https://github.com/MaderasAcme/scankey-v1a1.git
cd scankey-v1a1
npm i
npx expo start
```

## Deploy / Pages
El sitio web se publica en GitHub Pages (Source: **GitHub Actions**). Ver `docs/PAGES_SOURCE.md` para configuración de Pages y dominio.
